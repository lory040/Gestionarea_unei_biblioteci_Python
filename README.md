# Sistem de Management al Bibliotecii (Python OOP)
Acest proiect reprezintă o soluție completă pentru gestionarea unei biblioteci. Aplicația permite administrarea cărților, autorilor, utilizatorilor și proceselor de împrumut printr-o interfață interactivă.

#Concepte Tehnice Implementate

## Programare Orientată pe Obiect (OOP): Utilizarea claselor pentru a modela entități precum Carte, Autor, Utilizator și Biblioteca.
## Programare Funcțională: Utilizarea funcțiilor lambda, filter și map pentru căutarea și filtrarea eficientă a resurselor bibliotecii.
## Decoratori: Implementarea de decoratori pentru logarea execuției funcțiilor și verificarea permisiunilor de acces.
## Tratarea Excepțiilor: Gestiunea erorilor prin excepții personalizate (ex: CarteNegasitaException, UtilizatorInexistentException).
